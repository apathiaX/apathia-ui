## Props

| 参数    | 说明                                                                              | 类型            | 是否必传 | 默认值 |
| ------- | --------------------------------------------------------------------------------- | --------------- | -------- | ------ |
| src     | 原图片地址                                                                        | string          | 是       | -      |
| width   | 缩放图片，始终保持原图横宽比，和 height 属性两者传一个即可，若都传默认 width 生效 | string 、Number | 否       | -      |
| height  | 缩放图片，始终保持原图横宽比                                                      | string 、Number | 否       | -      |
| alt     | img alt 属性，图像描述                                                            | string          | 否       | 'img'  |
| preview | 是否可以预览                                                                      | boolean         | 否       | true   |

## Slots

| 名称  | 说明                   |
| ----- | ---------------------- |
| error | 图片加载失败的占位内容 |
